ft_printf



Ft_printf is a custom implementation of the

standard printf function in C.


It formats and prints output based on format

specifiers, handling different data types.

The function processes conversion flags and

modifiers to format strings, numbers, and

memory addresses efficiently.

The challenge is to recreate printf while ensuring

correct parsing, memory management, and performance.


The goal is to provide a reliable alternative to

printf with support for common specifiers.

The bonus extends the implementation by adding

flags, width, precision, and length modifiers.
