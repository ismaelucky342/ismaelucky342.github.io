push_swap



Push_swap is a sorting algorithm project

that challenges you to sort a stack of numbers

with the least number of operations.


The program uses two stacks and allows

three operations: push, swap, and rotate.

The goal is to find the most efficient way

to sort the numbers, minimizing the number of

operations and optimizing the algorithm.

The challenge is to implement a sorting algorithm

that works within time and space constraints.


The bonus extends the project by adding

the checkerr program, which verifies

whether a given sequence of operations is

valid and produces the correct sorted result.
