ft_irc



ft_irc is a custom implementation of an IRC server

that adheres to the RFC 1459 protocol specifications.


It allows multiple clients to connect, communicate

via channels, and exchange private messages.


The project focuses on handling concurrent connections,

managing user authentication, and implementing

basic IRC commands like JOIN, PART, PRIVMSG, and more.


The bonus includes advanced features such as

SSL/TLS encryption and additional IRC commands

to enhance the server's functionality and security.
