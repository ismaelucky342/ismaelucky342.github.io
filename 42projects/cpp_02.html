C++ Module 02



Objetivo del Proyecto

El objetivo del proyecto C++ - Module 02 es introducir conceptos fundamentales de la programación orientada a objetos en C++ mediante la implementación progresiva de clases que respeten la forma ortodoxa canónica (Orthodox Canonical Form), incluyendo:

  • Polimorfismo ad-hoc (a través de sobrecarga de operadores).
  • Sobre carga de operadores aritméticos, de comparación e incremento/decremento.
  • Uso de una clase Fixed que representa números de punto fijo como alternativa a los tipos enteros y de punto flotante tradicionales.

Objetivos específicos:

  • Diseñar clases en Canonical Form, implementando constructor por defecto, constructor de copia, operador de asignación y destructor.
  • Implementar una clase de punto fijo (Fixed), que permita conversiones entre enteros y flotantes, y que sea capaz de operar con operadores sobrecargados.
  • Aplicar estas clases en un contexto práctico creando una clase Point y una función bsp para determinar si un punto está dentro de un triángulo.
  • Fomentar buenas prácticas de diseño en C++98, incluyendo separación de código en .hpp y .cpp, manejo adecuado de memoria, y uso limitado del estándar de la biblioteca.
Este módulo es una parte clave del recorrido de aprendizaje de C++ en 42 y busca sentar bases sólidas en el uso de clases, sobrecarga y diseño orientado a objetos clásico.


✅ Exercise 00: My First Class in Orthodox Canonical Form

Objetivo: Crear una clase Fixed en forma ortodoxa canónica.

  • Implementar los 4 métodos canónicos: constructor por defecto, constructor de copia, operador de asignación y destructor.
  • Añadir métodos getRawBits() y setRawBits() para acceder al valor interno (entero) que representa el número de punto fijo.
✅ Exercise 01: Towards a more useful fixed-point number class

Objetivo: Hacer útil la clase Fixed.

  • Añadir constructores que acepten int y float como parámetros, y los conviertan al formato de punto fijo.
  • Implementar métodos toFloat() y toInt() para convertir de vuelta.
  • Sobrecargar el operador << para mostrar el valor como float con std::ostream.
Exercise 02: Now we’re talking

Objetivo: Añadir sobrecarga de operadores a la clase Fixed.

  • Sobrecargar operadores de comparación: >, <, >=, <=, ==, !=.
  • Sobrecargar operadores aritméticos: +, -, *, /.
  • Sobrecargar operadores de incremento y decremento (pre y post).
  • Añadir métodos min y max (estáticos) que devuelvan referencias al menor o mayor de dos objetos Fixed.
Exercise 03: BSP (Bonus - no obligatorio)

Objetivo: Aplicar la clase Fixed en un contexto más complejo.

  • Crear una clase Point con coordenadas x y y del tipo Fixed.
  • Implementar una función bsp(Point a, Point b, Point c, Point point) que devuelve true si point está dentro del triángulo abc (excluyendo bordes y vértices).
  • Usar álgebra o determinantes para comprobar si un punto está dentro de un triángulo (algoritmo BSP - Binary Space Partitioning).
