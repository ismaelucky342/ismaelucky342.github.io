MiniShell



Minishell is an implementation of a Unix shell

that allows executing commands and managing processes.


It uses an AST to parse and execute commands,

including logical operators and wildcards. The AST

structures execution into nodes, making it easier to

handle priorities and command combinations.


The goal is to create a functional shell that

supports input parsing, redirections, and pipelines.

The bonus extends the functionality with wildcards

and logical operators, enhancing command execution.
