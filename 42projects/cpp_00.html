Exercise 00: Megaphone

Description: The "Megaphone" exercise involves creating a C++

program that takes command-line arguments and converts them to

uppercase. If no arguments are provided, the program should

print a loud and annoying feedback noise message.

Objective:


->Handling command-line arguments.

->Converting characters to uppercase.

->Printing messages to the console.

Exercise 01: Phonebook

Description: The "Phonebook" exercise involves creating a C++ program to manage contact information.

The program allows users to add new contacts and search for existing ones.

Each contact consists of a first name, last name, nickname, phone number, and a secret.

Objective:


-> Handling user input and commands.

-> Managing and storing contact information.

-> Implementing a simple phonebook with add and search functionalities.

Exercise 02: Account Class

Description: This project implements a C++ Account class that simulates the basic behavior of a bank account.

The class handles operations like deposits and withdrawals and provides both individual and global account information.

Each transaction also displays a timestamp to track the operation time.

Objective:


-> Implement the Account class with deposit and withdrawal operations.

-> Track and display global statistics for all accounts.

-> Display account and transaction information with timestamps.

-> Create a test program to validate functionality and correctness.
